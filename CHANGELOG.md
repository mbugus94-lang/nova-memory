# Changelog

All notable changes to Nova Memory will be documented in this file.

## [2.1.1] - 2026-03-23

### Security & Maintenance
- Updated all dependencies to latest stable versions
- Updated FastAPI to 0.115.11 for latest security patches
- Updated cryptography to 44.0.0 for improved security
- Updated starlette to 0.46.1 for latest bug fixes
- Updated httpx to 0.28.0 for improved HTTP handling

## [2.1.0] - 2025

### Added
- Central server mode with enhanced features
- Swift context retrieval endpoint
- Auto-capture for chat interactions
- Collaboration spaces
- JWT authentication

### Changed
- Improved SQLite performance with WAL mode
- Enhanced Redis cache integration

## [2.0.0] - 2025

### Added
- Initial release of Nova Memory 2.0
- Multi-agent memory management
- SQLite persistence
- Optional Redis caching
- OpenClaw skill integration
